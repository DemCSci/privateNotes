# Changelog

## v1.8.10

- 重新发布正确的 GitHub Release 版本，避免 `v1.8.9` 因错误 tag 指向旧提交而生成 `1.8.8` 资产
- 保留 `1.8.9` 中的 MD5 兼容性修复

## v1.8.9

- 替换不稳定的 Hutool 16 位 MD5 计算逻辑，避免在 IntelliJ IDEA 2025 中触发 `ArrayIndexOutOfBoundsException`
- 旧 Git 兼容存储查找失败时不再影响私人注释可见性判断

## v1.8.8

- 增加 GitHub Release 自动发布流程
- 推送 `v*` tag 后可自动构建并上传插件 zip 到 GitHub Releases
- 补充仓库分发说明与更新记录文档

## v1.8.7

- 所有私人注释都支持悬浮弹出完整查看窗口，不再仅限于被省略的注释
- 悬浮查看窗口支持选中、复制、滚动

## v1.8.6

- 将悬浮注释容器替换为独立交互窗口，规避 IntelliJ IDEA 2025 中 hover popup 的交互限制
- 保持悬浮查看场景下的可交互性和稳定性

## v1.8.5

- 将悬浮注释内容切换为滚动编辑框样式的查看器
- 支持在悬浮查看窗口中滚动长文本

## v1.8.4

- 修复点击悬浮窗口时提前关闭的问题
- 提升悬浮窗口内文本选择与复制的可靠性

## v1.8.3

- 支持在悬浮窗口中选中和复制私人注释
- 增加 `Cmd/Ctrl+C`、`Cmd/Ctrl+A` 和右键菜单复制能力

## v1.8.2

- 修复缺失源文件或缺少本地字节内容时的读取异常
- 避免注释查询过程中出现运行时失败

## v1.8.1

- 增加路径前缀迁移结果报告
- 迁移结果支持滚动查看 migrated / skipped 明细

## v1.8.0

- 使用规范化绝对路径哈希作为主存储键
- 在注释文件中写入规范化源文件路径元数据
- 增加路径前缀迁移能力
- 保持对旧版本存储格式和旧 git 相关格式的兼容读取

## v1.7.1

- 使用 git 仓库根路径哈希与仓库相对路径哈希组合存储注释
- 非 git 仓库文件回退到规范化绝对路径哈希

## v1.7.0

- 通过源文件路径隔离注释存储，避免同名文件冲突
- 首次访问时自动迁移旧格式注释
- 改进多行替换后的注释行定位逻辑

## v1.6.4

- 增加悬浮弹层默认停留时间
- 关闭前重新检查指针位置，避免过早消失

## v1.6.3

- 悬浮弹层停留时间改为可配置

## v1.6.2

- 悬浮弹层支持滚动
- 鼠标可移动到弹层区域内查看内容

## v1.6.1

- 修复 IntelliJ IDEA 2025 中原有 tooltip 方式导致的崩溃

## v1.6.0

- 初步适配 IntelliJ IDEA 2025 平台
- 修复输入法候选框问题
- 优化高频路径上的性能问题
